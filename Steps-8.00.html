Importing Data and Load the Data 8

Raw File Dataset Link:https://www.kaggle.com/datasets/thedevastator/std-infection-rates-in-america-1996-2008?select=STD+Cases.csv


1.    Download the dataset from Kaggle and save it on your desktop

2.     Open Power BI Desktop

3.     Click on Get data button 

4.     Select  Text/CSV and click on connect

5.    Select the 'STD’ file and click on Open

6.     A preview of the data will be displayed; click on Load/Transform



 Load the Data

 Follow the file attached to customize the background for the report. This gives a better custom-looking feel to the report while also saving space by not using shapes.
