Create Line and Stacked Column Chart:



1.First, we are going to need a "Line and Stacked Column Chart" to use for our visual, so select that visualization and place it in the top left box in our report.

2. Next, we will select the values and measures that are going to be used in this chart. We will be using the "STD Cases" measure and "Calendar" value from the "Main Dataset" along with all of the Rate and YoY measures we created in this chart.

3. Selecting these measures and values doesn't mean they're going to the right places. Place "Calendar" in the shared axis, place " STD Cases," and rename it to " STD Cases " for this visual, then lastly, place all the YoY values into the line values; see the final screenshot for the renaming.

4. For the Title, simply rename the chart to "Cases By Year."

5. Now, for the formatting, just follow the GIF. It will give you all you need to know on how to format this chart. Keep in mind that the colors for " STD Cases" needs to be the main Light Green, the  " Rate Per 100k " needs to be White, " " Infection Rate " needs to be Purple, " Male " needs to be Yellow, the " Rate Per 100k " needs to be White, " Female " needs to be Light Blue " YoY " needs to be Orange as the KPIs.

6. Do the same thing as Step the First Chart, "Line and Stacked Column Chart," copy and paste Cases By Age and Cases By Disease.
