Create  Infection Rate and Infection YoY KPI Arrow:


  

1.   Double click on the Card Icon on the visualization bar.

2.   Card visual will display on the report

3.   Drag and drop the "Arrow YoY" column from the STD Case Table

4.   Configure the Format Infection Rate KPI Arrow

                              a. Click General Tab:

·         Horizontal: 4

·         Vertical:  574

·         Width: 44

·         Height: 39

·         Categories: Off

                              b. Data Label:

·     Color: Click

·         Color format by: Select “Field Value” from the list

·         Based on Field: Select from the list STD Cases “Color Arrow YoY” Column

·         Text size: 14  

·         Background: #000000

·         Transparency: 96%

c.    Boarder OFF
