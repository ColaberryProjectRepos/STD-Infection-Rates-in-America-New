Creating KPIs Cards 16

Creating the Total Cases Card is very similar to creating the slicers.




STEP 1: Create Total Case KPI

1.   Double-click on the Card Icon on the visualization bar.

2.   Card visual will display on the report

3.   Drag and drop the "Total Case " Measure from the STD Cases

4.   Configure the Format Total Cases KPI

 

a.    Click General Tab:

·         Horizontal: 0

·         Vertical:  532

·         Width: 182

·         Height: 106

·         Categories: Off

b.                Title: ON

·         Type: “Total Cases”

·         Font Color: White

·         Background Color: #70B0E0 Text Size: 12

·         Font Family: DIN

·         Background: Black (#000000)

·         Transparency: 15%

c.    Boarder ON

·         Boarder Color: White, 10% darker  

 

Step 2:  Create Rate Per 100k and Infection Rate  KPI

1.    Repeat (1 to 3 steps) to create Infection Rate

2.    Drag and drop the "Infection Rate " measure from the STD Case table

3.    Configure the Format %Fatalities

a.    Click General Tab:

·         Horizontal: 0

·         Vertical:  532

·         Width: 182

·         Height: 106

·         Categories: Off

b.    Title: ON

·         Type: “%Fatalities”

·         Font Color: White

c.    Background: 

·         Color: #70B0BE0

·         Text Size: 12

·         Font Family: DIN Background: Black (#000000)

·         Transparency: 15%

·         Boarder ON

·         Boarder Color: Color: Gray, 10% darker  

d.    Do the same thing as Step 1. Expect to do it for our "Total Cases and Rate Per 100k color white with Male and Female" Measure. This KPI will use Light Green (#919E89) inside the Male color (#FCB714) and Female color (#8dc0e6). Type: “%Fatalities”(Make sure to change the measure format to "Thousands.” This is very important!)

 

Step 3: Create Infection  YoY KPI  

   

1.   Double-click on the Card Icon on the visualization bar.

2.   Card visual will display on the report

3.   Drag and drop the "infection YoY" measure from the STD Case table

4.   Configure the Format Infection YoY KPI

 

A. Click General Tab:

·         Horizontal: 0

·         Vertical:  532

·         Width: 182

·         Height: 106

·         Categories: Off

         B. Title: ON

·         Type: “Infection YoY”

·         Font Color: White

         C.  Background: 

·         Color: #70B0E0

·         Text Size: 12

·         Font Family: Arial

·         Background: Black (#000000) 

        Boarder ON Transparency 15% Boarder Color White, 10% darker
