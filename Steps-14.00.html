Creating the main title and logo:


  • Insert a text box and write the main title: STD Infection Rate in America.
  • Insert Colaberry and STD Infection Rate in America virus logo. 

Create the Title

1.    Click on the Insert tab and "Text Box" Button on the ribbon to add a text box to the report.

2.    Type in the box ". Insert a text box and write the main title: STD Infection Rate in America."

a.    Font: Arial color White     

b.    Text Size:  28     

c.     Bold     

d.     Center   

e.    Background Color # 000000

f.     Fix the box Size:  Width: 1280, Height: 112

g.    Turn the Border ON and make the color = White, 10% darker and radius = 10 


Load the Logos

1.    Click the Insert tab from the panel.

2.    Select the image option and browse for the “STD Infection Rate in America virus logo” image from  your computer, 

3.    Repeat the same process to upload the Colaberry Logo. 

4.     Display both logos on the side of the Title as shown below.
